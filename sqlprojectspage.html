Project 1 - Washington State Electric Vehicle Analysis

In this SQL project, I leveraged the Washington State Department of Licensing's dataset on Battery Electric Vehicles (BEVs) and Plug-in Hybrid Electric Vehicles (PHEVs) to analyze and derive valuable insights. Using MySQL Workbench, I performed AD-HOC analysis to showcase key marketshare information within the dataset.

The dataset used for this project can be found at data.gov. There are over 150,000 total vehicles in this dataset. Some of the interesting columns include: Model Year, Make, Model and Electric Vehicle Type.
