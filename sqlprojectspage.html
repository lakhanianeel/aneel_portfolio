Project 1 - Washington State Electric Vehicle Analysis

Project 2 - Oklahoma Purchase Card Program Analysis

In this project, I conducted another analysis utalizing mysql.
The dataset used in this project can be found at catalog.data.gov/dataset/purchase-card-pcard-fiscal-year-2022-31015. It contains over 25,000 transactions.

This dataset contains information on purchases made through the purchase card programs administered by the state of Oklahoma and higher education institutions for the 2022 fiscal year.

The screenshot below shows most of the important columns.




The below query utilizes the GROUP BY clause and the SUM(AMOUNT) function to identify the top 20 agencies with the highest total purchase amounts.

Additionally, it calculates the percentage of each agency's expenditure in relation to the overall spending. This calculation is performed by dividing an agency's sum of purchase amounts by the total purchase amount of all agencies, which is obtained through a subquery using the SELECT SUM(AMOUNT) function and the FROM statement.




With this query shown below, I looked at transaction counts of MCC (Merchandise Category Code) descriptions, to find the most common merchandise categories and their corresponding transaction volumes.

This query groups the data by MCC_DESCRIPTION and calculates the count of transactions within each category using the COUNT(*) function. By leveraging the GROUP BY clause, I ensured that the results were aggregated by MCC_DESCRIPTION.




In this next and final query, I look more in detail into expenditure trends and patterns.

I use the COUNT(*) and SUM(AMOUNT) functions again as well as the MAX(AMOUNT) function which finds the maximum amount spent in a single transaction by each agency. This shows the highest individual expenditure for each agency.

The MIN(AMOUNT) function retrieves the minimum amount spent in a single transaction by each agency. This reveals lowest individual expenditure for each agency.

The AVG(AMOUNT) function calculates the average amount spent per transaction by each agency. This helps gives an understanding of the typical expenditure level for each agency.

Creating the code for this query demonstrates my ability to extract meaningful insights from financial datasets, analyze transaction data, and provide potentially actionable information for informed decision-making in financial analysis and resource allocation.
